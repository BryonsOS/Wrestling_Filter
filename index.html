One-time setup

This app searches YouTube with your own free API key (it stays in your browser — nothing is uploaded or downloaded anywhere).

  1. Go to Google Cloud Console → YouTube Data API v3 and click Enable (create a free project if asked).
  2. Go to Credentials → Create credentials → API key.
  3. Paste the key into Settings here.

The free quota (10,000 units/day ≈ 100 searches) is plenty — results are cached for 24 hours so browsing sections you've already loaded costs nothing.
